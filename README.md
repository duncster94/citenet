# CiteNet

Please see the wiki for more in-depth details.

## Usage

To install and run locally

```
$ npm install
$ npm start 
```

optionally, you can run it with `nodemon` so that any changes cause the server to restart automatically

```
$ npm install -g nodemon
$ nodemon app.js
```
